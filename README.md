Marketing Multi-Agent System

An AI-powered multi-agent marketing strategy generator built using n8n and OpenAI.
The workflow accepts a campaign brief and distributes the task among specialized AI agents:

Brand Strategy Agent
Content Strategy Agent
Paid Ads Strategy Agent
CMO Summary Agent

The outputs are merged into a structured executive marketing plan with actionable milestones.

Features:

Multi-agent architecture
OpenAI-powered specialists
Strategy aggregation
Structured JSON outputs
Automated 90-day action plans

Workflow Architecture:

Campaign Brief Input
